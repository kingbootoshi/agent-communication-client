This directory will contain sound effects for the VOID interface.

Expected files (to be generated with AI):
- void_ambient_loop.mp3 (Background ambient music)
- void_agent_message.mp3 (Sound when an agent sends a message)
- void_button_click.mp3 (Sound when clicking buttons)
- void_creation_complete.mp3 (Sound when character creation is completed)
- void_dm_message.mp3 (Sound when the DM sends a message)
- void_error.mp3 (Sound for errors)
- void_notification.mp3 (Sound for notifications)

Generate prompts for these sounds using the "Generate Sound Prompts" button in the UI.
